# GDAMamba: Global Difference-Aware Mamba for Hyperspectral Image Change Detection
This work has been accpeted by IEEE Transactions on Geoscience and Remote Sensing (TGRS) and can be downloaded [here](https://ieeexplore.ieee.org/abstract/document/11141454).

## Installation
You can directly refer to the [UNMamba](https://github.com/Preston-Dong/UNMamba) environment.

## Patch-wise apporach
You can directly use the [AIWSEN](https://github.com/creativeXin/AIWSEN) framework and replace the provided GDAMambapatch model with the existing one.
